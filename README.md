## 🎯 Про проєкт
Лабораторна робота №2 з розробки навчального контенту з інформатики. Створено повноцінний інтерактивний курс, що відповідає програмі МОН (поглиблений рівень). Проєкт містить теоретичний матеріал, різнорівневі практичні завдання, творчі проєкти, глосарій та інтерактивний тест на JavaScript.

## 🧠 Використання Генеративного ШІ
Увесь контент створено за допомогою ШІ та ретельно адаптовано під вікові особливості учнів:
* **ChatGPT (GPT-4o) / Claude:** Генерація теоретичного матеріалу, складання глосарію, розробка практичних завдань (кейсів), генерація коду для тесту (JS) та розмітки блок-схем (Mermaid).
* **Bing Image Creator (DALL-E 3):** Генерація графічного оформлення (обкладинка курсу).

*Приклад системного промпту, що використовувався:* "Дій як експерт з педагогіки. Згенеруй теоретичний матеріал у форматі Markdown про основи машинного навчання для 11 класу. Створи 4 тестових питання різних типів."

## 📂 Структура репозиторію
Проєкт має чітку ієрархію згідно з технічним завданням:
- `/theory` — теоретичний матеріал, приклади, глосарій.
- `/practice` — практичні завдання, лабораторна робота, творчий проєкт.
- `/tests` — інтерактивний тест (JS) та підсумкове оцінювання.
- `/resources` — посилання на зовнішні ресурси та згенеровані медіа.
- `/styles` та `/scripts` — CSS стилі та логіка для перевірки інтерактивного тесту.
